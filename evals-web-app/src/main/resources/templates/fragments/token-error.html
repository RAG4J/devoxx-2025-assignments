Authentication Token Required

Your authentication token has expired or is invalid. This is required to communicate with the AI service.

To refresh your token:
  1. Open the main Thymeleaf Agent application (usually at http://localhost:8080)
  2. Navigate to the Token Management page
  3. Generate a new token or refresh your existing token
  4. The new token will be automatically shared with this evaluation application
  5. Try running your evaluation again
